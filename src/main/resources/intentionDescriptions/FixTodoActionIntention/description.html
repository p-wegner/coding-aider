Provides a quick way to fix TODO comments using AI assistance.

This intention allows you to automatically resolve TODO comments in your code using AI-powered suggestions. It immediately sends the TODO comment and surrounding context to the AI for processing without additional configuration.

Example TODO comment that can be fixed:


// TODO: Add input validation for the user's age
public void processAge(int age) {
    // Implementation needed
}

The AI will analyze the TODO comment and surrounding code to suggest an appropriate implementation.
